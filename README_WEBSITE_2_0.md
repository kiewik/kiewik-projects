# Kiewik Projects Website 2.0

A static, responsive and bilingual homepage for Kiewik Projects.

## Includes

- English and Polish interface
- direct Horizon AI Chrome Web Store link
- real Horizon AI screenshots
- project roadmap cards
- privacy and support sections
- responsive layout
- accessible image preview
- no analytics, tracking libraries or external frameworks

## Publishing

GitHub Pages already publishes this repository from `main` and `/(root)`.
Replace the root `index.html`, upload the `assets` folder and the accompanying
root files, then commit the changes.

Do not remove the existing `horizon-ai/privacy/` folder.
