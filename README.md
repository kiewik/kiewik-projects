# Kiewik Projects — GitHub Pages

Gotowy pakiet do publikacji jako repozytorium `kiewik-projects`.

## Docelowe adresy

- Strona główna: `https://kiewik.github.io/kiewik-projects/`
- Polityka Horizon AI: `https://kiewik.github.io/kiewik-projects/horizon-ai/privacy/`

## Publikacja

1. Utwórz publiczne repozytorium `kiewik-projects`.
2. Wgraj całą zawartość tego folderu do katalogu głównego repozytorium.
3. Wejdź w `Settings → Pages`.
4. Ustaw `Deploy from a branch`, gałąź `main`, folder `/(root)`.
5. Zapisz i zaczekaj na publikację.

Do formularza Chrome Web Store wklej bezpośredni adres polityki Horizon AI, a nie samą stronę główną.
