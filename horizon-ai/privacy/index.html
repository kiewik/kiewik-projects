Horizon for Claude Privacy Policy

Effective date: 23 July 2026

This legacy address is retained for compatibility with earlier Chrome Web Store listings. The current canonical address is https://kiewik.github.io/kiewik-projects/horizon-for-claude/privacy/.

Summary: Horizon does not read prompts or conversation content, does not use advertising or analytics telemetry, and does not sell user data.

1. Scope

This policy applies to the Horizon for Claude Chrome extension developed by Kiewik Projects.

2. Data the extension handles

Horizon reads only the usage-limit information made available in the Claude.ai interface that is necessary to provide the extension’s core features, such as current 5-hour and weekly usage, reset timing and related status indicators.

Horizon does not read, collect or store conversation messages, prompts, files attached to conversations or the generated response content.

3. Local storage and Chrome Sync

  • Usage readings, local history and forecasts are stored locally in Chrome on the user’s device.
  • Display preferences and selected settings may be stored using Chrome Sync so they can follow the user across signed-in Chrome browsers.
  • A local installation loaded as an unpacked extension has a separate storage space from the Chrome Web Store installation.

4. External transmission

Horizon does not send usage readings, history, forecasts or conversation content to Kiewik Projects or to external analytics services. The extension includes no advertising telemetry.

5. Permissions

  • Host access / scripting: used only to display and update Horizon on Claude.ai.
  • Storage: used for local history, forecasts and settings.
  • Side panel: used to display the full statistics view.
  • Alarms and notifications: used for optional limit-related alerts.

6. Data deletion

Users can clear stored readings and forecasts from the extension’s data controls. Removing the extension also removes its local Chrome storage. Chrome Sync data can be cleared through Chrome account settings.

7. Third-party services

Horizon operates on Claude.ai but is an independent extension. It is not affiliated with, endorsed by or produced by Anthropic.

8. Changes to this policy

Material changes will be published on this page with an updated effective date.

9. Contact

Questions about privacy can be sent to kiewikprojects@gmail.com.
